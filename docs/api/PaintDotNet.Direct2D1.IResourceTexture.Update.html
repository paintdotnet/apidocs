Updates the specific resource texture inside the specific range or box using the supplied data.

void Update(int dimensions, in ResourceTextureExtents? extents, ExtentPtr<byte> data, ResourceTextureStrides dataStrides)

Parameters

dimensions int

The number of dimensions in the resource texture. This must match the number used to load the texture.

extents ResourceTextureExtents?

The extents of the data to be updated, or null to update the whole texture.

data ExtentPtr<byte>

The data to be placed into the resource texture.

dataStrides ResourceTextureStrides

The number of bytes between the start of rows (for 2D textures) and planes (for 3D textures).
