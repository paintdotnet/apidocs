The base class for implementing a custom effect in Direct2D. The custom effects system in PaintDotNet.Direct2D1 handles many things automatically, such as just-in-time registration and auto-generation of the registration XML.

public abstract class CustomEffect<TProperties> : DeviceEffect<TProperties>, ISerializable, IDeviceEffect, IDeviceImage, IDirect2DResource, IFactoryProperty, IDeviceEffectProperties, IObjectRef, IIsDisposed, IDisposable, IInternalImpl where TProperties : CustomEffectProperties, new()
