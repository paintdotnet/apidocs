Remarks

An implementation of this interface is provided when using Open().

To implement this yourself, such as to capture the data provided by Tessellate(ITessellationSink, in Matrix3x2Float?, float?) for custom purposes, create a class that derives from RefTrackedObject and that also derives from this interface.
