CreateMesh()

Create a mesh that uses triangles to describe a shape.

IDeviceMesh CreateMesh()

Returns

IDeviceMesh

The new mesh object.

Remarks

To populate a mesh, use the Open() method to obtain an ITessellationSink. The sink may be filled with triangles manually, or by using the Tessellate(ITessellationSink, in Matrix3x2Float?, float?) method.
To draw the mesh, use the device context's FillMesh(IDeviceMesh, IDeviceBrush) method.
