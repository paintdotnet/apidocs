CreateBitmapBrush(IDeviceBitmap, in BitmapBrushProperties?, in BrushProperties?)

Creates a bitmap brush, the input image is a Direct2D bitmap object.

IBitmapBrush CreateBitmapBrush(IDeviceBitmap bitmap, in BitmapBrushProperties? bitmapBrushProperties = null, in BrushProperties? brushProperties = null)

Parameters

bitmap IDeviceBitmap

The bitmap to use as the brush.

bitmapBrushProperties BitmapBrushProperties?

A bitmap brush properties structure, or null. If you set this parameter to null, the brush defaults to Clamp for clamping, and Linear for interpolation mode.

brushProperties BrushProperties?

A brush properties structure, or null. If you set this parameter to null, the brush uses an opacity of 1.0f and an identity transform matrix.

Returns

IBitmapBrush

The new bitmap brush object.
